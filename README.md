## ⚠️ 安全声明

操作空中机器人具有危险性！请严格遵守安全规范！首次试飞请务必寻求有自稳模式飞行经验的飞手协助。

---

## 🛠 技术栈

- **飞控**: PX4 v1.11.0 (CUAV V5)
- **机载电脑**: Intel NUC
- **深度相机**: Intel RealSense D435i
- **SLAM**: VINS-Fusion
- **规划**: Ego-Planner
- **ROS**: Noetic (Ubuntu 20.04)
